Projet E-Cockpit dashboard OBDII BLE

Application Android • Bluetooth • OBD2

Description

Pouvoir étandre l'affichage des données embarqué de toutes voiture (>1997) sur une application Android via une connection bluetooth OBD2.

Fonctionnalités

  • Fonction 1 : Affichage des données capteurs
  • Fonction 2 : Visualisation en temps réel
  • Fonction 3 : Gestion des alertes
